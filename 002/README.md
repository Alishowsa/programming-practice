# 📚 Sistema de Cadastro de Alunos

## Descrição

Projeto desenvolvido em JavaScript com o objetivo de praticar lógica de programação, manipulação de arrays, objetos e funções.

## Funcionalidades

- Cadastrar alunos
- Listar alunos
- Mostrar alunos aprovados
- Calcular média da turma
- Buscar aluno pelo nome
- Encontrar o aluno com maior nota
- Ordenar alunos por nota

## Tecnologias

- JavaScript
- Node.js

## Estrutura

```
cadastro-alunos/
│
├── app.js
└── README.md
```

## Como executar

Clone o repositório.

```bash
git clone URL_DO_REPOSITORIO
```

Entre na pasta.

```bash
cd cadastro-alunos
```

Execute.

```bash
node index.js
```
